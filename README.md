# VMath Beta

#### VIETNAMESE
**VMath** là phần mềm toán học miễn phí cho mọi người.
*Bao gồm các tính năng:*
- Nhập xuất dữ liệu
- Các hàm toán học
- 2 Ngôn Ngữ: Tiếng Việt & Tiếng Anh

#### ENGLISH
**VMath** is a free math program for everyone.
*Featuring:*
- Input / Output Data
- Math Functions
- 2 Lauguages: Vietnamese & English
